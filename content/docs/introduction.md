# Introduction

GoStarter is a production-focused Go SaaS boilerplate. It includes the systems most teams rebuild over and over: authentication, billing, admin routes, background jobs, and docs-ready UI patterns.

## What is included

- `chi` HTTP routing and middleware.
- Postgres with `pgx`, `sqlc`, and `goose`.
- Server-rendered UI using `templ`, Datastar, and Tailwind.
- JWT API auth plus web session cookies.
- Stripe Checkout, Billing Portal, and webhook synchronization.
- Asynq workers for email and async tasks.

## Who this is for

GoStarter fits indie founders and product teams who want explicit architecture, typed SQL, and a codebase that is meant to be extended, not thrown away.
